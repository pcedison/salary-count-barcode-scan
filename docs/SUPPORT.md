# 技術支援指南

## 系統概述

員工薪資計算系統是一個完整的考勤與薪資管理解決方案，提供以下核心功能：

- 員工資料管理
- 條碼掃描打卡
- 考勤記錄管理
- 薪資自動計算
- 報表生成與匯出
- 系統設置與管理

## 系統架構

```
前端 (React + TypeScript)
    ↓
API 層 (Express.js)
    ↓
業務邏輯層 (薪資計算、考勤處理)
    ↓
資料存取層 (Drizzle ORM)
    ↓
資料庫 (PostgreSQL/Supabase)
```

## 核心功能說明

### 1. 員工管理模組

**功能**：
- 新增、編輯、刪除員工資料
- 員工資料加密保護
- 部門和職位管理
- 時薪設定

**相關檔案**：
- `shared/schema.ts` - 資料模型定義
- `server/routes.ts` - API 路由
- `client/src/pages/AttendancePage.tsx` - 前端頁面

### 2. 考勤管理模組

**功能**：
- 手動考勤登記
- 條碼掃描打卡
- 考勤記錄編輯
- 工時自動計算

**相關檔案**：
- `client/src/pages/BarcodeScanPage.tsx` - 條碼掃描頁面
- `client/src/hooks/useAttendanceData.ts` - 考勤資料處理

### 3. 薪資計算模組

**功能**：
- 基本薪資計算
- 加班費計算（按日累計）
- 勞健保扣款
- 薪資記錄管理

**計算邏輯**：
```javascript
// 基本薪資 = 工作時數 × 時薪
基本薪資 = 總工作時數 × 基本時薪

// 加班費計算（按日）
每日加班費 = 當日加班時數 × 時薪 × 加班倍率
月加班費 = Σ(每日加班費)

// 總薪資
總薪資 = 基本薪資 + 加班費 - 勞健保扣款
```

### 4. 報表模組

**功能**：
- 月薪資報表
- 考勤統計
- 加班統計
- 資料匯出 (Excel, PDF)

## 常見使用情境

### 情境 1：新增員工

1. 進入「考勤登記」頁面
2. 點擊「新增員工」按鈕
3. 填寫員工基本資料
4. 設定時薪和部門
5. 儲存員工資料

### 情境 2：條碼打卡

1. 進入「條碼掃描打卡」頁面
2. 員工使用掃碼槍掃描身份證條碼
3. 系統自動識別員工並記錄打卡時間
4. 顯示打卡成功訊息

### 情境 3：薪資計算

1. 進入「薪資計算」頁面
2. 選擇計算年月
3. 選擇員工或批量計算
4. 系統自動計算薪資
5. 檢視和匯出薪資報表

## 資料流程

### 打卡資料流程

```
條碼掃描 → 身份識別 → 考勤記錄 → 工時計算 → 薪資計算
```

### 薪資計算流程

```
考勤資料 → 工時統計 → 加班時數計算 → 基本薪資計算 → 扣款計算 → 淨薪資
```

## 系統限制與注意事項

### 效能限制

- 單次匯出最大記錄數：10,000 筆
- 同時線上使用者：100 人
- 檔案上傳大小：10MB
- 資料庫連接數：最大 10 個

### 資料限制

- 員工姓名：最長 100 字元
- 身份證號：固定格式驗證
- 考勤日期：支援 2020-2030 年
- 時薪範圍：0-9999 元

### 業務規則

- 每日工作時數上限：24 小時
- 加班時數計算：超過 8 小時視為加班
- 薪資計算週期：按月計算
- 資料保存期限：3 年

## 疑難排解步驟

### 問題分類

#### 1. 登入問題
- 檢查管理員密碼
- 確認瀏覽器 Cookie 設定
- 清除瀏覽器快取

#### 2. 資料顯示問題
- 檢查資料庫連接
- 確認 API 服務運行
- 檢查網路連接

#### 3. 計算錯誤
- 檢查系統設定
- 確認考勤資料完整性
- 重新計算薪資

#### 4. 效能問題
- 檢查系統資源使用
- 清理舊資料
- 重新啟動服務
