# 員工薪資管理系統 - 完整交付包

## 專案概述
員工薪資管理系統是一個完整的企業級薪資管理解決方案，具備考勤追蹤、薪資計算、報表生成和數據匯出功能。

### 核心功能
- **考勤管理**: 員工打卡、加班時數追蹤
- **薪資計算**: 自動化薪資計算，支援加班費、津貼、扣款
- **報表生成**: CSV匯出、列印功能
- **歷史記錄**: 薪資記錄查詢和管理
- **系統設定**: 薪資規則、扣款項目配置

## 技術架構

### 前端技術棧
- **框架**: React 18 + TypeScript
- **路由**: Wouter
- **狀態管理**: TanStack Query
- **UI組件**: shadcn/ui + Tailwind CSS
- **表單處理**: React Hook Form + Zod

### 後端技術棧
- **運行環境**: Node.js + Express
- **數據庫**: PostgreSQL (支援Supabase)
- **ORM**: Drizzle ORM
- **認證**: Passport.js (本地策略)
- **會話管理**: Express Session

### 部署技術
- **開發環境**: Vite
- **生產環境**: Express靜態文件服務
- **資料庫**: Supabase (雲端) 或本地PostgreSQL

## 核心功能詳解

### 薪資計算邏輯
- **基本薪資**: 28590元（可配置）
- **津貼**: 福利津貼、住宿津貼
- **加班費計算**: 
  - OT1 (1-2小時): 基本時薪 × 1.34
  - OT2 (超過2小時): 基本時薪 × 1.67
- **扣款項目**: 勞保費、健保費、服務費、宿舍費等

### 數據安全
- **身份證號加密**: 凱薩密碼加密存儲
- **會話管理**: Express Session + PostgreSQL存儲
- **管理員權限**: PIN碼驗證系統

## 部署指南

### 環境要求
- Node.js 18+
- PostgreSQL 12+ 或 Supabase帳戶
- 4GB+ RAM
- 10GB+ 磁碟空間

### 快速部署步驟

1. **解壓縮項目文件**
```bash
tar -xzf employee-salary-system-final.tar.gz
cd employee-salary-system
```

2. **安裝依賴**
```bash
npm install
```

3. **配置環境變數**
```bash
cp .env.example .env
# 編輯 .env 文件，設置數據庫連接
```

4. **數據庫設置**
```bash
# 使用Supabase (推薦)
npm run db:setup:supabase

# 或使用本地PostgreSQL
npm run db:setup:local
```

5. **啟動應用**
```bash
# 開發環境
npm run dev

# 生產環境
npm run build
npm start
```

### 環境變數配置
```env
# 數據庫設置 (Supabase推薦)
DATABASE_URL=postgresql://[username]:[password]@[host]:[port]/[database]
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your-supabase-anon-key

# 會話安全
SESSION_SECRET=your-session-secret-key

# 管理員設置
ADMIN_PIN=1234

# 系統設置
NODE_ENV=production
PORT=5000
```

## 操作手冊

### 管理員功能
1. **員工管理**: 新增、編輯、刪除員工資料
2. **薪資設定**: 配置基本薪資、津貼、扣款項目
3. **數據匯出**: 批量下載薪資記錄
4. **系統備份**: 自動和手動備份功能

### 使用者功能
1. **考勤打卡**: 上下班時間記錄
2. **薪資查詢**: 查看個人薪資記錄
3. **報表下載**: CSV格式薪資明細

## 維護指南

### 數據備份
系統提供自動備份功能：
- **每日備份**: 自動執行，保留7天
- **週備份**: 保留4週
- **月備份**: 保留12個月

手動備份：
```bash
npm run backup:manual
```

### 系統監控
監控要點：
- 數據庫連接狀態
- 備份任務執行狀況

### 故障排除

#### 常見問題

1. **數據庫連接失敗**
   - 檢查DATABASE_URL設置
   - 確認數據庫服務運行狀態
   - 驗證網路連接

2. **薪資計算錯誤**
   - 檢查考勤記錄完整性
   - 驗證薪資設定參數
   - 查看錯誤日誌

3. **檔案匯出問題**
   - 確認瀏覽器允許下載
   - 檢查磁碟空間
   - 驗證數據完整性
